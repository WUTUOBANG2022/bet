### 管理员操作

#### 添加新比赛

使用API创建比赛：
```bash
curl -X POST "https://your-backend-url.railway.app/matches/" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "英超：曼联 vs 利物浦",
    "team_a": "曼联",
    "team_b": "利物浦",
    "odds_a": 2.1,
    "odds_b": 1.9,
    "odds_draw": 3.2,
    "start_time": "2024-01-20T15:00:00"
  }'
```

#### 查看所有比赛

```bash
curl "https://your-backend-url.railway.app/matches/"
```

#### 结算比赛

```bash
curl -X PUT "https://your-backend-url.railway.app/matches/1" \
  -H "Content-Type: application/json" \
  -d '{
    "score_a": 2,
    "score_b": 1,
    "winner": "A",
    "status": "finished"
  }'
```
