# 📱 分享给朋友的使用指南

## 快速开始

### 第一次使用

1. **打开应用**
   ```
   访问：https://your-app-url.railway.app
   ```

2. **创建账户**
   ```
   点击「立即注册」
   填入用户名、邮箱、密码
   完成注册
   ```

3. **登录账户**
   ```
   输入用户名和密码
   点击「登录」
   ```

4. **初始余额**
   ```
   每个新账户初始有 ¥1000
   可用于下注
   ```

## 使用流程

### 1️⃣ 查看比赛

- 点击「比赛」查看所有可用的比赛
- 每场比赛显示两支队伍和赔率
- 赔率越高，风险越大，收益越高

### 2️⃣ 下注

1. 选择你要下注的比赛
2. 点击下注按钮
3. 选择支持的队伍或平局
4. 输入下注金额
5. 查看潜在收益
6. 点击「确认下注」

**示例：**
- 下注 ¥100 在赔率为 2.5 的球队
- 潜在收益 = ¥100 × 2.5 = ¥250

### 3️⃣ 等待结果

- 比赛进行时显示「进行中」
- 比赛结束时会自动结算
- 赢了自动加入余额
- 输了下注金额会扣除

### 4️⃣ 查看我的下注

- 点击「我的下注」查看历史记录
- 可以筛选「全部」、「待结算」、「赢利」、「亏损」
- 查看下注详情和结果

### 5️⃣ 钱包管理

- **查看余额**：点击导航栏看当前余额
- **存款**：需要更多钱进行下注
- **提现**：将赢利提现
- **交易记录**：查看所有存提和下注记录

## 赔率说明

### 什么是赔率？

赔率是你下注获胜后的收益倍数。

```
下注金额 × 赔率 = 潜在收益

例如：
- 下注 ¥100
- 赔率 1.8
- 赢了就得 ¥180
- 净收益 ¥80（收益 - 本金）
```

### 赔率高低

| 赔率 | 风险 | 收益 | 建议 |
|-----|------|------|------|
| 1.2-1.5 | 低 | 低 | 稳定选手 |
| 1.5-2.5 | 中 | 中 | 平衡玩法 |
| 2.5-5.0 | 高 | 高 | 冒险玩家 |

## 下注策略

### 初学者建议

1. **从小金额开始**
   - 先用 ¥10-50 熟悉系统
   
2. **选择低赔率**
   - 风险更小，成功率更高
   
3. **分散下注**
   - 不要把所有钱压在一场比赛
   
4. **观察赔率变化**
   - 赔率会随着下注变化

### 风险管理

⚠️ **重要提示：**
- 这是模拟下注，不涉及真实金钱
- 仅供娱乐和学习
- 任何实际赌博都存在风险
- 合理安排下注金额

## 常见问题

### Q: 忘记密码怎么办？
A: 联系应用管理员重置密码。

### Q: 为什么不能下注？
A: 可能原因：
- 余额不足
- 比赛已结束或已取消
- 网络问题

### Q: 下注后能取消吗？
A: 不能，下注后不可取消。请谨慎操作。

### Q: 什么时候结算？
A: 比赛结束时自动结算，立即看到结果。

### Q: 如何提现？
A: 点击「钱包」→ 「提现」→ 输入金额（仅限模拟）。

### Q: 钱什么时候到账？
A: 模拟系统中立即到账。

## 联系方式

遇到问题可以：
- 直接告诉应用管理员
- 在应用内查看帮助
- 检查是否网络连接正常
